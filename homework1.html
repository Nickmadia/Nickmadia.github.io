Homework 1: Basic Statistical Concepts

Title

Introduction to Descriptive Statistics

Research

In this homework, we explore the basics of descriptive statistics, including mean, median, mode, and standard deviation. These concepts are essential for summarizing data.

Code

# Python code to calculate basic statistics
import numpy as np

data = [1, 2, 3, 4, 5, 6, 7, 8, 9]

mean = np.mean(data)
median = np.median(data)
std_dev = np.std(data)

print(f"Mean: {mean}, Median: {median}, Standard Deviation: {std_dev}")
